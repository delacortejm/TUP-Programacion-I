# Programación I

Materia correspondiente al 1er año de la carrera **Técnico Universitario en Programación (TUP)** — UTN Facultad Regional Resistencia.

## Contenido

La materia se centra en los fundamentos de la programación utilizando el lenguaje **C**, abarcando temas como estructura de un programa, tipos de datos, operadores, funciones de entrada/salida, estructuras de control, entre otros.

## Estructura del repositorio

```
├── examenes/     → Parciales y finales
├── práctica/     → Ejercicios resueltos por unidad
├── programa/     → Programa de la materia
└── teoría/       → Material teórico y apuntes
```

## Compilación

Para compilar cualquier ejercicio en C:

```bash
gcc archivo.c -o archivo
./archivo
```
